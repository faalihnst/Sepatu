									Web toko sepatu olahraga
1. Penjelasan
	Web ini menyediakan layanan beli sepatu. Kita membuat toko ini untuk mempermudah orang orang yang ingin membeli sepatu sehingga tidak perlu repot repot lagi untuk
pergi ke toko fisik untuk membeli sepatu. Dapat berupa pre-order atau ready stock. Umumnya web ini menjual sepatu dengan brand Nike, Air Jordan, dan Adidas tapi tidak
menutup kemungkinan bagi web ini untuk menjual brand sepatu lain tergantung permintaan pasar.

2. Fungsi
	Web ini menyediakan fungsi memesan sepatu dan juga login. Para pengguna wajib untuk login sebelum memesan. Pada tiap akun terdapat alamat dan juga ukuran sepatu pengguna
sehingga saat pemesanan tidak perlu repot lagi untuk menyantumkan alamat dan juga ukuran sepatu. Terdapat fungsi penjualan dan pembelian di web ini. Ada juga fungsi yang melayani pre-order
untuk pelanggan.
